Good and Bad Web Design

In this assignment we will be looking at more examples of good and bad web design. To see more examples of good web design, please vist Smashing Magazines examples of good design . You should read the pages there on best practices. Look at the examples there and pick 5 examples from the site or anywhere else of Web sites where you like the design. I want you to present the pages you like and a description of a minimum of three reasons why you like them in the following format:

The Good

Example of Web Page With Good Design

Apple Computers

I like this site, because it is simple and balanced. I like the fact that the designer was not afraid to use a lot of white space on the main page. It uses a basic color palate consistently throughout the entire site. I like the fact that the home page has only a few links to very logical categories. Even though there are only a few links on this homepage, you are only a few clicks from most things you would want to find on the site.

First Example of Web Page With Good Design

Tesla

I like this site because:

  • navigation at the top (including home page link) with simple structure of: product name, image, links to product
  • not too much noise (no extraneous details), all links are related to product(s)
  • font/color pops, unified

Second Example of Web Page With Good Design

Github

I like this site because:

  • dark theme (easier on eyes)
  • navigation at top, includes search bar (the page is longer than I'd like so this is nice)
  • font/colors pop (including images), sleek

Third Example of Web Page With Good Design

Warpfrog

I like this site because:

  • darker theme, pretty, good logo
  • simple navigation at top of page
  • external links for additional news/info

Fourth Example of Web Page With Good Design

Wolf & Wood

I like this site because:

  • about section
  • unified font, colors contrast
  • text is centered; lots of margin space

Fifth Example of Web Page With Good Design

Nexus mods

I like this site because:

  • navigation at top, with search bar
  • relatively dark, plenty of margin space
  • neatly organized
